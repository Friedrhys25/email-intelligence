# Setup Prompt

## Context

Inbox Intelligence is a TypeScript backend service that will run locally and on Render.

## Goal

Create the project foundation.

## Requirements

- Node.js and TypeScript setup.
- Express app with `/health`.
- Environment validation.
- Pino logger.
- Test setup.
- Folder structure that supports Clean Architecture.
- Scripts for development, build, start, test, and lint.

## Deliverables

- Project configuration files.
- Source folder structure.
- Health route.
- Logger and env config.
- Initial tests.

## Acceptance Criteria

- App starts locally.
- Build passes.
- Tests pass.
- Missing required env vars fail clearly.
- Folder structure matches docs.

### Prompt Template

```text
Using the System Prompt, create the Inbox Intelligence project setup. Configure Node.js, Express, TypeScript, environment validation, logging, tests, scripts, and the documented folder structure.
```
